Markdown 语法

发表于 2021-10-21 更新于 2021-11-06 阅读次数：

1、标题 （文字底部用 = 和 - 标记一级和二级标题）

二级标题

比如一级标题

2、换行 （段落的换行是行尾两个以上的空格 + 回车）

比如这是一行
这又是一行

3、字体 （markdown可以使用三种字体）

  • _斜体_ 斜体文本
  • __粗体__ 粗体文本
  • ___粗斜体___ 粗斜体

4、分割线 （一行中用三个以上的星号、减号、底线来建立分割线，行内不能有其他东西，可以有空格）

  • ***/* *
  • ------
  • ______

5、删除线 （文字两端加两个英文波浪线~~）

比如 ~~删除~~ 删除内容

6、下划线（用HTML的标签<u>文字</u>）

带下划线的文字

7、脚注 （格式 [^脚注内容]）

创建脚注格式类似这样 [^mark]。
[^mark]:这个是脚注里面的内容，会追加到最后

8、列表

  • 无序列表 使用* 、+ 、- 后面加个空格
  • 有序列表 数字加.加空格
    1. 第一个
    2. 第二个
  • 列表嵌套（只需在子列表中的选项前面添加个四个空格即可）
    比如有序嵌无序
  1. 第一级
    1. 一级1
      1. 三级
        1. 四级，哦 一般就到三级，
    2. 一级2
  2. 第二大级

9、区块（段落开头用>, 后面紧跟一个空格）

> 这是一块
> 这个是第二块，换行是用双空格 + 回车的

  • 区块也可以嵌套（多个>>）

    最外层

    1. 区块里面第一行
    2. 区块里面第二行

      >> 里面第一层

      >>> 里面第二层

10、代码

  1. 段落里的代码片段，可以用反``把它包起来

    比如print()函数

  2. 代码区块，用 ``` 代码 ``` 包起来

11、链接

  • 普通链接 [链接名称](链接地址)
    比如，这是个链接
  • 高级链接， 用变量作为网址
    这是多次要使用的一个变量a, 没错就是变量， 格式是[变量][a]
    这是多次要使用的一个变量b，是的就是变量， 格式是[变量][b]
    [a]:https://www.baidu.com
    [b]: https://www.baidu.com

12、图片

![alt 属性文本](图片地址)
![alt 属性文本](图片地址 "可选地址")

  • 开头一个感叹号
  • 接着一个方括号，里面放图片的替代文字
  • 接着一个普通括号，里面放图片网址，后面选填图片‘title’文字

13、表格（｜分隔不同单元格，- 分隔表头和其他行）

  • -: 右对齐
  • :- 左对齐
  • :-: 居中对齐
    name age
    tom 23
    jarry 55

    14、支持的HTML标签

  • <b><i><em><br><kbd><sup><sub>
  • <b>粗 类似 _粗体_
  • <i>、<em>斜体 类似 _斜体_
  • <kbd>Ctrl shift
  • <sub> 你是我儿子
  • <sup> 我是你爹

15、特殊

  • 888 文本加粗用前后两个**
  • 显示 ‘*’ 用转义符号, 正常的**
